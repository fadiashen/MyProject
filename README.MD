# XONIX
**2.5D арканоід для ОС Windows**

##Опис задачi
**Сенс гри — захопити >= 75% поля. По полю хаотично бігають кульки. Швидкiсть кульок приблизно 20% ширини поля за 1 секунду (буде уточнюватися в процессi розробки). Якщо вони зіткнуться з лінією, яку гравець веде, або гравець зiткнеться зi своею ж лiнiею, - втрачаємо життя та становимося в позицiю з якої почалася гра теперiшньго рiвня(середина ближнього ребра поля). На початку гри даеться три життя. Нам треба захопити >= 75% поля для переходу на наступний рівень. На кожному наступному рiвнi кiлькiсть кульок збiльшуеться на одну та добавляеться одне життя до iснуючих. Коли кiлькiсть кульок зростає настiльки, що вони займають 10% грального поля, — то гра вважаеться пройдена i гравцю виводеться на екран повiдомлення про перемогу. Пiсля повiдомлення про перемогу при натисканнi будь якої клавiшi йде перехiд до меню(варiанти меню: "Нова гра", "Вихiд"). На першому рiвнi присутня одна кулька. **
